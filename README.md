# NayePankh Foundation Hub

## Overview
NayePankh Foundation Hub is a Streamlit-based NGO Management System designed to manage volunteers, campaigns, donations, and reports efficiently. The platform provides a centralized dashboard that helps organize social welfare activities and improve operational transparency.

## About NayePankh Foundation
NayePankh Foundation is a non-profit organization focused on education, healthcare, community development, and social welfare. Through volunteer initiatives, awareness campaigns, donation drives, and outreach programs, it empowers underprivileged communities, supports social causes, and creates a positive impact on society.

## Features
- Volunteer Management
- Campaign Management
- Donation Tracking
- Search and Filter Options
- Report Generation
- CSV Export Functionality
- Interactive Dashboard

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- HTML/CSS

## Project Structure
- `NayePankh.py` – Main Application
- `database.py` – Database Operations
- `reports.py` – Reports and Analytics
- `logo.jpeg` – Foundation Logo
- `campaign_bg.jpeg` – Campaign Banner
- `volunteer_bg.jpeg` – Volunteer Banner

## Installation

```bash
pip install -r requirements.txt
streamlit run NayePankh.py
```

## Objective
The objective of this project is to simplify NGO operations by providing a digital platform for managing volunteers, campaigns, donations, and reports while improving efficiency, transparency, and data management.
